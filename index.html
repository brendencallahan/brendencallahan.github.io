Hi, I’m Brenden, musician turned
Fullstack Developer.

After attending the University of Idaho where I was playing with, and writing for, so many wonderful musicians, I felt it was time for a career change. Since then, I've built this portfolio, using HTML, CSS, and JavaScript, a small business' landing page, using React and Tailwind, a stock portfolio webapp, using Flask and SQL, and a multiplayer connet 4 game, using React and ... Along with these projects I've also completed several online courses most notably Harvard's Intro to Computer Science. (Programs written for this class are available on my GitHub.)

In my free time I enjoy riding my bike, playing saxophone, and relaxing with video games. I have always been interested in technology/music and would love to discuss either with you!
